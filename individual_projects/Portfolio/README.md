# Tkinter Portfolio

A portfolio project to explore my favorite code projects i have made this year. 

## Description

This is a portfolio project that displays my favorite projects i have made in my programming class. it provides a simple discription and works with an intuitive GUI powered by tkinter.

### Dependencies

Tkinter library must be installed for this project to run


### Executing program

to run program, press run in VS code, or another codepsace that allowed display enviorments,and a window should pop up. from there the code should instruct you on what to do
