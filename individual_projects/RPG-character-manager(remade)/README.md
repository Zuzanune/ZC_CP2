# Project Title

A character tracker and creator for dnd. create characters, save them, and compare their stats in a bar chart. also includes a random character generator.
compatable with 5e. anything else I am not sure.

## Description

to use this program, run main.py. you will be prompted to create a character or load characters from a file. if you choose to create a character, you will be prompted to enter the characters stats.
you can also choose to have the stats randomly generated. if you choose to load characters from a file, you will be able to view the characters and their stats. you will compare the characters stats in a bar chart
if you find something in your dnd game that does not appears here(spells, special attacks, etc.) edit character Skills and add the new information there.

## Getting Started

### Dependencies

* You must have the pandas library and matplotlib library installed to run this program.
* to install, open a new terminal and copy/paste this code into it
```
pip install pandas
```
and this code
```
pip install matplotlib
```

### Executing program

* How to run program
* run in vs-code. 
* click run button. 
* be on main page.

## Help
If any errors come up, there is no way to regain any information that you entered in since the last save. characters are saved after creation and editing. 
To avoid errors don't try and break the program. enter in correct values and avoid placing spaces everywhere.
The bar graph does not appear when run on a chromebook. sorry.
